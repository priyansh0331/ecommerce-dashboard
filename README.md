# 📊 E-Commerce Sales Analytics Dashboard

An end-to-end data analytics project using a real Amazon India sales dataset (~129K orders). The project covers data cleaning with Python, analytical queries with SQL, and an interactive dashboard built in Power BI.

---

## Dashboard Highlights

| Visual | Insight |
|---|---|
| 📈 Monthly Revenue Trend | Revenue across April–June 2022 |
| 🏆 Top Categories | Best-selling product types by revenue |
| 🗺️ Revenue by State | Regional sales performance map |
| 🔄 Order Status Breakdown | Delivered vs Cancelled vs Returned |
| 👥 B2B vs B2C | Business vs consumer order comparison |
| 📦 Cancellation Rate | Which categories cancel the most |

## Dataset

**Source:** [Amazon Sales Dataset – Kaggle](https://www.kaggle.com/datasets/thedevastator/unlock-profits-with-e-commerce-sales-data)

- ~129,000 orders from Amazon India
- Date range: April – June 2022
- Columns: Order ID, Date, Category, Amount, Status, Ship State/City, B2B, Sales Channel

## Setup & Usage

### 1. Clone the repo

```bash
git clone https://github.com/priyansh0331/ecommerce-dashboard.git
cd ecommerce-dashboard
```

### 2. Download the dataset

Download from [Kaggle](https://www.kaggle.com/datasets/thedevastator/unlock-profits-with-e-commerce-sales-data) and place it at:
```
data/ecommerce_sales.csv
```

### 3. Install dependencies & clean the data

```bash
pip install -r requirements.txt
python clean_data.py
```

This generates `data/ecommerce_sales_clean.csv` with:
- Parsed dates and derived time columns (month, week, day)
- Simplified order status labels
- Removed nulls and junk columns
- B2B/B2C flag and standardised category names

### 4. Open in Power BI

1. Open `dashboard.pbix` in **Power BI Desktop**
2. Go to **Home → Transform Data → Data Source Settings**
3. Update the path to point to your local `ecommerce_sales_clean.csv`
4. Click **Refresh** — all visuals will update automatically

### 5. Explore the SQL queries

Open `ecommerce_analysis.sql` in any SQL editor. To run against the CSV in SQLite:

```bash
# Install sqlite-utils
pip install sqlite-utils

# Load CSV into SQLite
sqlite-utils insert ecommerce.db ecommerce_sales data/ecommerce_sales_clean.csv --csv

# Run queries
sqlite3 ecommerce.db < ecommerce_analysis.sql
```

## Key Findings

- **April** had the highest revenue (~₹28.8M), followed by May (~₹26.2M) and June (~₹23.4M)
- **Kurta** and **Set** are the top-selling categories
- **Maharashtra**, **Karnataka**, and **Tamil Nadu** lead in regional sales
- Cancellation rates vary significantly by category
